# Hospital System

## Project Description
A basic Hospital Management System written in Java. It uses Object-Oriented Programming to manage patients, doctors, and bills. It also uses dynamic user input and exception handling to prevent the program from crashing.

## Exceptions Implemented
* **IllegalArgumentException**: Used to stop the program if a user types a blank name or a negative fee.
* **NumberFormatException**: Used to catch errors if a user types words instead of numbers for the fees.
* **Exception**: A general catch block for any other random errors.
* **Finally block**: Used to close the Scanner and release resources safely.
